This repository is a minimal scaffold for the BokTalent project.

What is included:

- Next.js (App Router) TypeScript scaffold
- TailwindCSS + PostCSS
- ESLint + Prettier (config placeholders)
- Vitest for unit tests
- Playwright for e2e tests
- Basic role routes: /player, /scout, /school, /admin
- `.env.example` with required keys
- GitHub Actions workflow placeholder

Quick start

1. Install dependencies

```bash
cd /path/to/boktalent
npm install
```

2. Start dev server

```bash
npm run dev
```

3. Run unit tests

```bash
npm run test:unit
```

4. Run e2e (Playwright must be set up and browsers installed)

```bash
npm run test:e2e
```

Replace placeholders in `.env.example` with real values before connecting external services.
